# Tesla Q2 2025 EDA & Financial Analysis

This project analyzes Tesla’s financial and operational performance for Q2 2025 using exploratory data analysis (EDA) and visualization tools in Python.

## 🔍 Project Highlights

- Revenue, Profit, EPS, and Margin Analysis
- Stock Price Trends and Market Reaction
- Vehicle Delivery and Production Metrics
- Visual exploration of earnings trends vs. past quarters

## 📊 Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📄 Report

A full HTML and/or PDF version of the notebook is available in the repository. The visuals explain Tesla’s business narrative through data.
